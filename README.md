# Typescript种子项目 
使用routing-controllers框架，底层基于koa，自动注入路由、session、请求参数等信息

### 热启动命令
```
gulp watch
```
### 直接启动方式

#### 编译
```
gulp compile

```

#### 开发环境启动
```
node dist/boot/development.js  
```

#### 测试环境启动
```
node dist/boot/testing.js
```

#### 生产环境启动
```
node dist/boot/production.js  
```
